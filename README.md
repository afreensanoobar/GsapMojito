 Mojito Website – Velvet Pour

A modern, animated Mojito-themed website built using GSAP and ScrollTrigger for smooth animations and interactive effects.

Features

GSAP animations on load, hover, and scroll

Responsive design for desktop and mobile

Interactive buttons, menus, and images

Clean and modern “Velvet Pour” theme

Technologies Used

HTML5

CSS3 / SCSS

JavaScript (ES6+)

GSAP & ScrollTrigger for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.
